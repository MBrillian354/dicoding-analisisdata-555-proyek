## Requirements

To run the code, we will need to install:
- pandas~=2.2.0
- matplotlib~=3.8.2
- seaborn~=0.13.2
- streamlit~=1.30.0
- Babel~=2.14.0


You can install the libraries with this code:

```
pip install -r requirements.txt
```

## Usage

Run streamlit app
```
streamlit run dashboard.py
```
